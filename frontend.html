Frontend


En desarrollo web, frontend se refiere a la parte del sitio o aplicación con la que los usuarios interactúan directamente. Incluye todo lo que los usuarios ven y con lo que interactúan en sus navegadores, como el diseño, la estructura, los botones, y los contenidos. Se basa en tecnologías como HTML, CSS y JavaScript para construir interfaces visuales y garantizar una experiencia de usuario fluida y atractiva.


Fundamentos:


  • HTML: Define la estructura de la página web mediante elementos como encabezados, párrafos, enlaces, imágenes, listas, etc.

  • CSS: Controla el diseño de la página, como colores, tipografías, márgenes, espaciado, layout y el estilo general.

  • JavaScript: Permite crear efectos dinámicos en una página web, validar formularios, modificar elementos HTML/CSS en tiempo real, y gestionar eventos como clics, desplazamientos o entradas del teclado.
